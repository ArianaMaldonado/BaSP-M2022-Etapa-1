Trackgenix

What we do

Trackgenix is a web system to ease the registration of the hours worked by each employee in each project and work team.

Functionalities

Timesheet

In this section, both the PM and the employee are able to complete and visualize their own personal timesheet and the current status of every project they are involved in. However, the timesheet of each employee and each project has to be approved by the PM.

Reports

This web system provides you the possibility of generating reports segmented by date for each employee or project. The Admin role can also generate an informative report of each task the employees have been working in individually.

Resource management

At the end of every week, Human Resources section must receive the form with the timesheet data in order to manage the mensual payment of their employees as appropriate. You only need to generate the reports you need.

Multiple roles

Each role has different functionalities depending on whether you are an Employee, Project Manager (PM), Admin or SuperAdmin. At the same time, within PM's functionalities, there is the possibility of assigning a specific role to each employee, such as Developer, QA, TL, etc.
